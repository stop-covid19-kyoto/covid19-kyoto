# 京都府 新型コロナウイルス 総合情報サイト

![](https://github.com/stop-covid19-kyoto/covid19-kyoto/workflows/production%20deploy/badge.svg)

[![京都府 新型コロナウイルス 総合情報サイト](https://raw.githubusercontent.com/stop-covid19-kyoto/covid19-kyoto/development/static/ogp.jpg)](#)

当サイトは新型コロナウイルス感染症 (COVID-19) に関する最新情報を提供するために、京都府が開設したものです。

京都府による公式情報と客観的な数値をわかりやすく伝えることで、京都府にお住まいの方や、京都府内に拠点を持つ企業の方、京都府を訪れる方が、現状を把握して適切な対策を取れるようにすることを目的としています。
